# 🐶 어디가냥? 같이가개!

> 해당 레포지토리는 frontend 레포지토리입니다.

## 🖥️ 프로젝트 소개

어디가냥?같이가개!는 반려동물과 함께할 수 있는 공간을 쉽게 찾고 공유하는 플랫폼 입니다.

## 📅 개발기간

25.07. ~ 25.09.

## ⚙️ 주요 기능

- jwt 토큰을 이용한 소셜 로그인
- 카테고리, 키워드 별 장소 검색
- 인기순, AI 추천을 통한 장소 추천 기능
- 각 장소들의 정보 및 사용자 리뷰 및 별점 열람
- 사용자 & 반려동물 프로필 등록, 수정, 삭제, 기능
- 리뷰 작성, 수정, 삭제, 신고 기능
- AI 플래너를 통한 나들이 계획 생성 기능
- 반응형 UI를 통한 모바일 및 태블릿 뷰포트 지원

## 📁 디렉토리 구조

- components : 컴포넌트 폴더 (각 페이지 / 공용에 필요한 컴포넌트가 위치합니다.)
- configs : 설정 폴더 (상수, 옵션 등)
- hooks : 커스텀 훅 폴더
- lib : 함수 모음 폴더
- screens : 페이지 폴더 (메인화면, 로그인 화면, 마이페이지 등)
- stores : zustand스토어 폴더
- styles : 추가 스타일 폴더
- types : 타입 선언 폴더
